# AMS_Temp_Master

## Overview
- AMSの温度測定基板
- スレーブ側
- 2023年度10月時点で使用

## Requirement
### Development
- Kicad 7.0
### Boards
- Arduino UNO
### Libraries

## Usage
- Arduinoマイコンボードとして使用
- I2C でスレーブ側からデータ取得
- CAN の使用方法はArduino UNO で CAN BUS Shield を使用した時と同じ
